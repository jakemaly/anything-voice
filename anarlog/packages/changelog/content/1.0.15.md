## Onboarding

- You can now skip sign-in and start using the app immediately
- Microphone permission copy is simplified and clearer
- System restarts during setup are deferred until onboarding completes
- Removed the unnecessary system audio restart prompt that interrupted the flow
- Auth callback input now shows an example placeholder

## Recording

- Audio file uploads are back — drag or pick files directly from the note input area
- Upload progress is now shown when converting audio
- Recordings can be deleted from the audio player
- `Listen` button has improved visual states for idle, recording, and finalizing

## Audio Player

- Native context menu with play/pause and seek controls on the timeline
- Text selection no longer interferes with player controls

## Notes

- Images from notes now appear in session preview cards
- Fixed tab indentation breaking across split task lists

## UI

- Toast notifications moved to the top of the screen
- Left sidebar layout and session preview cards refined
- New app icon replaces the wobbling head in chat
- Removed the pulsing red dot on the active tab
- Solid backgrounds restored for dialogs and modals in popup windows
- Upload menu styling fixed for new meetings

## Platform

- Migrated to Vite 8
- `Cmd+Q` now uses a double-press to quit instead of hold-to-quit
