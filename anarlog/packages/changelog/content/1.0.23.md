## Transcript & Session

- Transcript panel toggle redesigned as a border tab labeled "Transcript" or "Live", replacing the old chevron arrows
- Press Escape to collapse an expanded transcript panel
- Delete recording button added to the post-session audio area
- Expanding the transcript panel now automatically moves floating chat to the right panel to avoid overlap
- Enhanced note tabs now show the template name instead of generic labels like "Summary"

## Speech-to-Text

- AquaVoice added as a new STT provider (Avalon V1 model, batch only)

## Calendar & Navigation

- Connecting an OAuth calendar provider now returns to the correct originating view instead of always going back to the calendar
- Profile menu calendar entry now routes directly to the calendar settings tab

## UI Polish

- Ad-hoc recording button hidden when already viewing the active live session
- Profile menu dividers removed for a cleaner look
- Tab bar height and spacing refined for better alignment
