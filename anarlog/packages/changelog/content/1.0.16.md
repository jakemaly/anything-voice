## Transcript

- You can now copy speaker-labeled transcript text directly from the Transcript tab
- Recording can be deleted from the session overflow menu without deleting the entire session

## Images

- Right-click context menu on image attachments with copy and reveal-in-Finder actions
- Resize handles stay visible inside the image frame instead of clipping outside
- Images without stored width default to 80% of the editor width
- Selected images show a blue border instead of a text-selection highlight
- Resize handles are hidden during full-document selection to reduce visual noise

## Notes

- Session preview cards now prefer the AI summary over raw memo content
- Reverted the empty-session save guard that was preventing some sessions from persisting

## Calendar

- Calendar sync no longer fails when one provider is unreachable
