## Calendar

- Improved sync feedback — shows "Syncing" status with spinner in the month header
- Events sort by all-day status and start time for clearer day views
- Provider chevrons only appear on hover to reduce visual clutter
- Unattached notes are visually distinguished from calendar events
- New Calendar tab in Settings for managing permissions and providers
- Google Calendar connection can be launched directly from the empty provider row
- Google Calendar badge updated from "Internal Use Only" to "Beta"

## Notes & Recording

- Cmd/Ctrl+N creates a plain empty note; "Start New Meeting" is now a separate action
- Floating Listen button restored inside empty notes
- Start recording command available from empty pages
- Recording indicator dot added to resume listening button
- Fixed false mic-stopped alerts and restored auto-stop

## Chat

- Chat accessible from the tab bar during pre-listening notes
- Composer caps editor height and scrolls internally for long drafts
- Send button stays visible even with long messages

## Transcription

- Manual speaker assignment for transcript segments
- Various transcript rendering improvements

## UI

- Tab shortcut badges stay readable while holding Cmd
- Tab action buttons (pin/close) moved to the trailing edge
- Unfiy settings into single tab & use sidebar as navigator

## Security & Distribution

- Fixed HTML injection in advanced search results
- DMG installers are now notarized and stapled on macOS
